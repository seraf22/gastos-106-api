## 📋 Quick Command Reference

```bash
# Development (Local)
cd src/Casa106.Web && npm run dev    # Frontend on :5173
dotnet run                            # API on :7210

# Build for Production  
cd src/Casa106.Web && npm run build  # React build
dotnet publish -c Release             # .NET build

# Docker testing
docker build -t casa106-api .
docker run -p 8080:8080 \
  -e DB_HOST=... \
  -e DB_PORT=... \
  casa106-api

# Git
git add .
git commit -m "feat: production deployment ready"
git push origin main                  # Triggers auto-deploy!
```

## 🎯 What Happens After You Push

```
YOU: git push origin main
	 │
	 ├─► GitHub receives code
	 │
	 ├─► Vercel: Detects React/Vite
	 │   └─► Auto-builds & deploys frontend
	 │       └─► ✅ Live in ~2 minutes
	 │
	 └─► Render: Detects Dockerfile
		 └─► Auto-builds & deploys backend
			 └─► ✅ Live in ~10 minutes
```

Both deploys happen **automatically** without your intervention!
